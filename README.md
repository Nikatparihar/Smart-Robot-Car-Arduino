# 🤖 Smart Robot Car using Arduino Nano

An Arduino Nano-based Smart Robot Car capable of operating in both **Obstacle Avoidance Mode** and **Bluetooth Control Mode**.

## 📌 Features

- Obstacle Avoidance using HC-SR04 Ultrasonic Sensor
- Bluetooth Control using HC-05
- Servo Motor for obstacle scanning
- LCD Display for status
- DC Motor control using L298N Motor Driver

## 🛠️ Components Used

- Arduino Nano
- HC-SR04 Ultrasonic Sensor
- HC-05 Bluetooth Module
- Servo Motor (SG90)
- L298N Motor Driver
- 16×2 LCD
- DC Motors
- Robot Chassis
- Battery Pack

## 💻 Software

- Arduino IDE
- Embedded C
